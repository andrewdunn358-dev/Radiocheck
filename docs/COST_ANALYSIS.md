## SECTION 4: SHOULD YOU GET PAID?

### The Question
You've built something worth £200,000-500,000 for essentially free. Should you (or the organisation) charge for this or pay yourselves?

### Options

#### Option A: Completely Free Service
| Pros | Cons |
|------|------|
| Maximum accessibility | No sustainability |
| Easy to fund (grants) | No income for maintenance |
| Clear charitable purpose | Burnout risk |

**Verdict**: Good for pilot phase, not sustainable long-term

#### Option B: Free to Users, Funded by Grants
| Pros | Cons |
|------|------|
| Users pay nothing | Grant dependency |
| Can pay for staff/development | Time-consuming applications |
| Professional sustainability | Funding gaps possible |

**Verdict**: Most appropriate model for veteran charity

#### Option C: Freemium Model
| Pros | Cons |
|------|------|
| Basic free, premium paid | Complexity |
| Sustainable income | May exclude vulnerable users |
| Investment attractive | Ethical concerns |

**Verdict**: Not recommended for mental health charity

#### Option D: NHS/Commissioned Service
| Pros | Cons |
|------|------|
| Stable funding | Long procurement process |
| Large user base | Bureaucracy |
| Professional credibility | Loss of independence |

**Verdict**: Long-term goal, not immediate

### Recommended Model

**Grant-Funded Charity with Modest Salaries**

| Role | Time | Annual Cost |
|------|------|-------------|
| Project Lead (You?) | Part-time (20hrs/week) | £15,000-25,000 |
| Clinical Supervisor | 10 hrs/month | £6,000-10,000 |
| Admin Support | Part-time | £10,000-15,000 |
| **Total Staff Costs** | | **£31,000-50,000** |

Plus running costs (~£3,000/year) = **Total: £34,000-53,000/year**

This is fundable through:
- Armed Forces Covenant Fund Trust
- Veterans Foundation
- National Lottery
- Corporate sponsors

**Key Technical Concepts**
-   **Twilio Voice SDK**: Used for browser-to-phone calling. Relies on a client-side  object that registers with Twilio using a short-lived JWT.
-   **Twilio JWT**: A JSON Web Token generated on the backend () using the Account SID, API Key SID, and API Key Secret. It grants the frontend device permission to make calls.
-   **TwiML (Twilio Markup Language)**: XML-based instructions returned by the backend () that tell Twilio how to handle a call (e.g., dial a number, play a message).
-   **Socket.IO**: Used for signaling for both internal WebRTC calls and the live text chat.
-   **Environment Variable Management**: Critical for integrations like Twilio. Mismatched or incorrect variables on the hosting platform (Render) were the root cause of most Twilio issues.
